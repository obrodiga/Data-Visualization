Руководство по использованию программы

Данная программа предназначена для визуализации и сравнения различных методов обработки сигналов. Изначально необходимо обработать данные с использованием встроенного математического аппарата, либо, при наличии, с использованием внешнего математического аппарата. Входные данных должны состоять из массива файлов *.txt, и для каждой из точек должно быть равно число файлов с измерениями.
После обработки формируется несколько, в зависимости от числа точек, файлов с расширением *.txt, которые в дальнейшем можно визуализировать в основном окне программы. Возможно отобразить значения в виде окружностей для относительного сравнения, и в виде графиков с абсолютными числами.
